คณะวิทยาศาสตร์และเทคโนโลยี สาขาวิทยาการคอมพิวเตอร์

รับสมัครนักศึกษา (ปริญญาตรีภาคจันทร์-ศุกร์ และภาคเสาร์-อาทิตย์)

หลักสูตรวิทยาการคอมพิวเตอร์ คณะวิทยาศาสตร์และเทคโนโลยี มหาวิทยาลัยราชภัฏวไลยอลงกรณ์ ในพระบรมราชูปถัมภ์ รับสมัครนักศึกษา TCAS 68 (ปริญญาตรี จันทร์-ศุกร์ และภาคเสาร์-อาทิตย์) ตั้งแต่วันนี้เป็นต้นไป ดูรายละเอียดและกรอกใบสมัครได้ที่นี่

สิ่งที่คุณจะได้เรียนในวิทยาการคอมพิวเตอร์

Technology Stack แต่ละชั้นปีเรียนอะไร

ชั้นปี ภาคปฏิบัติ ภาคทฤษฎี
1 การเขียนโปรแกรมพื้นฐาน เช่น C Python รวมไปถึงการพัฒนา Web Frontend เช่น HTML CSS JS, Bootstrap, Basic PHP รวมถึงหัดใช้งาน git service Algorithm, Data Structure, Computer Architecture, Operating System, System Analysis and Design
2 การเขียนโปรแกรมที่ซับซ้อนเพิ่มขึ้นอย่างภาษา Object Oriented เช่น ตระกูล java รวมไปถึงฐานข้อมูลต่างๆ ที่อยู่ในความนิยมทั้งรูปมีโครงสร้างตระกูล SQL และไร้โครงสร้าง NoSQL รวมทั้งใช้งานเครื่องมือ Full Stack Framework ที่เป็นที่นิยม Network, Data Communication, Software Engineering, Security
3 ทำระบบที่มีการนำ Backend และ Frondend มารวมกัน ทั้งในรูปแบบ Web Application ในกรณีศึกษาต่างๆ , การพัฒนา Mobile Application ด้วยเครื่องที่มีมีความทันสมัย, Internet Of Thing หรือแม้กระทั่ง Desktop Application ระเบียบวิธีวิจัย การประยุกต์ร่วมกับชุมชน วิทยาการคอมพิวเตอร์กับการบริการชุมชน
4 ระบบ Infrastucture on cloud ต่างๆ ซึ่งมีอยู่ด้วยกันหลายค่าย การ deploy ด้วย Docker และ Kubernetes รวมไปถึงความรู้พื้นฐานเกี่ยวกับ data science และปัญญาประดิษฐ์ ความรู้พื้นฐานเกี่ยวกับ data science และปัญญาประดิษฐ์ และวิชาเลือกเสรี
